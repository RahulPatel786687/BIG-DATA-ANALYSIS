*DESCRIPTION*: 
This project demonstrates Big Data Analysis using PySpark, focusing on distributed data processing and analytical operations using a real-world sample dataset. The objective of this project is to understand how large datasets can be processed efficiently using Apache Spark’s Python API.

Tools & Technologies Used
Python – Core programming language
PySpark – Python API for Apache Spark used for big data processing
Apache Spark – Framework for scalable and distributed computing
Jupyter Notebook – Development environment for coding and execution
Dataset Information

The project uses a publicly available online dataset (tips dataset) containing restaurant billing and tipping information. Since the dataset is loaded directly from an online source, there is no requirement to store a separate CSV file in the project repository.

Project Workflow

The analysis begins by creating a Spark Session, which acts as the entry point for running PySpark applications. The dataset is then imported into a Spark DataFrame using schema inference and header configuration.

Key operations performed in the project include:

Counting total records in the dataset
Calculating average bill values
Grouping and analyzing customer data by day
Filtering high-value transactions
Performing category-based analysis such as customer spending patterns

The project also demonstrates partitioning and distributed processing concepts, showing how PySpark can handle analytical tasks efficiently.

Platforms & Accessibility

This project can be executed on multiple platforms, making it flexible and easy to access:

Jupyter Notebook / VS Code – Run locally after installing Python and PySpark
Google Colab – Execute online without heavy local setup
Apache Spark Environments / Cloud Platforms – Compatible with systems supporting Spark workloads

This project highlights the practical use of PySpark for scalable data analysis, making it suitable for learning big data concepts and understanding distributed computing workflows.
